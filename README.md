📊 Dashboard Financeiro Pessoal - Power BI

Este projeto foi desenvolvido com o objetivo de transformar um controle financeiro pessoal em um dashboard interativo utilizando Power BI.

Embora eu já tivesse trabalhado com Power BI em ambiente profissional, este é o meu primeiro projeto voltado para uma necessidade do meu próprio dia a dia. A ideia surgiu da seguinte pergunta: "Por que não utilizar o Power BI para analisar minhas finanças pessoais?"

✨ Funcionalidades
Acompanhamento de receitas e despesas;
Comparativo entre entradas e saídas;
Evolução financeira ao longo do tempo;
Indicadores (KPIs) para análise rápida;
Filtros por período e categoria;
Dashboard totalmente interativo.
🛠️ Tecnologias utilizadas
Power BI
Power Query
DAX
Modelagem de Dados
⚠️ Importante

Os dados disponibilizados neste repositório são 100% fictícios e foram gerados com auxílio de IA.

Os dados reais utilizados no meu controle financeiro não foram publicados, preservando minha privacidade. A estrutura do projeto, no entanto, é exatamente a mesma, permitindo que qualquer pessoa utilize este modelo como base para seu próprio controle financeiro.

📥 Como utilizar
Faça o download do projeto.
Abra o arquivo .pbix no Power BI Desktop.
Caso deseje, substitua a base de dados fictícia pelos seus próprios dados.
Personalize categorias, métricas e visuais conforme sua necessidade.
💡 Objetivo

Além de organizar minhas finanças pessoais, este projeto também serviu para praticar conceitos importantes de:

Modelagem de dados;
Power Query;
Medidas em DAX;
Criação de dashboards interativos;
Storytelling com dados.
